**Decisions:**
- React SPA instead of Astro (dashboard interactivity needs)
- Mapbox GL JS for maps (client preference, not Leaflet)
- ChargePoint SOAP/XML not REST (their API is SOAP v5.0)
- Demo login bypasses API auth (CORS issue with cross-origin JWT)
- Crons intermittently 403'd by ChargePoint's Cloudflare — Sync Now buttons as backup
- Station IDs remapped from CSV EVSE IDs to ChargePoint stationIDs via name matching

**Known Issues:**
- Cron syncs intermittently blocked by ChargePoint's Cloudflare (403) — manual Sync Now buttons work reliably
- Mapbox token must be plain text env var in Pages (not encrypted)
- VITE_API_URL must be plain text in Pages env vars
- Demo login is client-side only — real JWT auth built but CORS blocks cross-origin login
- 10 stations couldn't be name-matched initially, resolved by full reimport from API
